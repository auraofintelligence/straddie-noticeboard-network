Brief

What this prototype is saying

    Boundary

    Supposed data, not consent

    Pipeline

    Boring files, flexible screens.

    The data layer stays simple so each business, artist, club or service can build its own look around it.

    Screen adapters

    The same notice fits different shapes.

    Theme calendar

    Scheduled meaning, not random content.

    Local and global themes can become opt-in publishing prompts. Each entity can tick what fits its values each month or quarter, ignore what does not fit, and add its own island-specific themes.

    Global prompts

    Local prompts

    Agent pipelines

    Ready S.E.T. media agents turn updates into safe public files.

    The co-op media layer can help entities draft, check, approve and render notices without forcing every local business or artist to become technical.

    Device location IDs

    Every screen needs a name, place and fallback mode.

    The markdown only becomes useful when devices around the island know where they are, what shape they are, what assets they have locally, and what to do offline.

    Markdown contract

    Frontmatter is the shared language.

    Repo files

    Sample feeds and device manifests.

    Straddie Noticeboard Network is a standalone public noticeboard pipeline prototype.

    Real feeds need consent, source checks, expiry dates, cultural care and privacy review before publication.
